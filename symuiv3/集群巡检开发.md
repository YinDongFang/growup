### Echarts使用记录

1. options
```javascript
{
      backgroundColor: 'rgba(207, 232, 252, 0.3)',  // 背景色， 可以取transparent
      grid: {     // 边界边框
        left: 0,
        right: 0,
        top: 0,
        bottom: 0
      },
      tooltip: {    // 提示器
        trigger: 'axis',
        position: (pt: any) => [pt[0], '10%'],
        formatter: (params) => {return ''}
      },
      xAxis: {      // 横轴定义
        type: 'time',
        show: false,
      },
      yAxis: {      // 纵轴定义
        type: 'value',
        max: this.max,
        show: false,
      },
      series: [{      // 序列数据
        name: this.name,
        type: 'line',
        showSymbol: false,
        hoverAnimation: false,
        areaStyle: {},
        data: this.data
      }]
}
```
